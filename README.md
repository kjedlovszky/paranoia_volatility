# Code for the preprint Jedlovszky, Corlett & Yon, 'Subjective Volatility, Learning and Paranoia' (2024)

Link to the preprint is available [here](https://osf.io/preprints/psyarxiv/sre9y_v1)


The data used for the analysis is available [here](https://osf.io/jkqhe/?view_only=339e1babb2cb44078b48d9e8bc2a9ec4)


The repository contains scripts for preprocessing, model fitting (plot_RLmodel.m), simulations (simulate_RLmodel.m) and data analysis (data_analysis_anovas.py). The code for the Rescorla-Wagner model and simulations was adapted from [freely available Matlab code](https://exp.socsci.ru.nl/hannekedenouden/RLtutorial/HtmlTutorial/RLtutorial_codeNdata.zip)
